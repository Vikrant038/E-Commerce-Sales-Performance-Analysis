# Screenshots & demo assets

The main `README.md` and your LinkedIn/portfolio posts need a visual. Drop assets here and link them from the README.

## What to capture (playbook A2)
- [ ] **`dashboard.png`** — one clean screenshot of the dashboard with KPIs + a chart visible. Embed this at the top of the README.
- [ ] **`demo.gif`** — optional short GIF of you changing a filter and the charts updating (use Kap, ScreenToGif, or LICEcap).
- [ ] **2-min Loom video** — record once the app is deployed; paste the link in the README and `docs/INSIGHTS.md`. People hire what they can *see* working.

## How to take the screenshot
1. Run the app: `streamlit run streamlit_app.py`
2. Open http://localhost:8501, let it load fully.
3. Capture the viewport (macOS: `Cmd+Shift+4`, then drag).
4. Save as `dashboard.png` in this folder.
5. In `README.md`, the image tag already points to `docs/screenshots/dashboard.png` — once the file exists it renders automatically.
