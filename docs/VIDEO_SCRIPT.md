# 🎥 2-Minute Demo Script — E-Commerce Sales Performance

A natural, conversational Loom script (~320 words ≈ 2:00 at a relaxed pace). Talk like you're showing a friend, not reading a paper. Text in **[brackets]** = what to show on screen.

> Before you record: open the live dashboard, reset filters, close extra tabs. Use [Loom](https://loom.com) — webcam bubble + screen. One take is fine; energy beats polish.

---

### [0:00 · webcam / title slide]
"Hey! I'm Vikrant. In the next two minutes I'll show you how I turned a pile of messy e-commerce data into a dashboard a business owner can actually *use* to make decisions. Let's jump in."

### [0:15 · GitHub repo + architecture diagram]
"The raw data came from two systems — a CRM and an ERP. Six messy tables, inconsistent types, the usual headache. So I built a SQL data warehouse using a **Medallion** design: raw data lands in *Bronze*, gets cleaned in *Silver*, and ends up in a clean *Gold* layer modeled as a star schema. Twelve SQL scripts handle everything from exploration to ranking to segmentation."

### [0:40 · live dashboard — top KPIs]
"And here's the fun part — the live dashboard. Right at the top: **29 million euros in revenue, 11.7 million in profit — a 39.8% margin** — across nearly 28,000 orders. And all of it is interactive…"

### [0:55 · click a filter — pick one country or category]
"…watch: I'll filter to just one market, and *every* chart and number updates instantly. No SQL, no spreadsheets — anyone on the team can explore this themselves."

### [1:10 · open the Insights tab]
"But data only matters if it tells you what to *do*. So I built an Insights tab. For example — bikes are **96.5%** of revenue, which is a risk. But here's the gold: **accessories earn a 63% profit margin versus 39% on bikes**, and they're barely 2% of sales. So there's a clear move — push accessories as add-ons, and profit jumps."

### [1:35 · scroll the insight cards]
"One more: just **37% of customers buy more than once, but they drive 77% of revenue** — so retention beats chasing new buyers. Every insight comes with a recommended *action* — like a mini consultant baked right into the app."

### [1:50 · webcam / outro]
"That's the whole project — clean data engineering plus an interactive, decision-ready dashboard. The code and live link are right below. Thanks for watching — let's talk!"

## Quick timing cheat-sheet
| Time | Beat | Screen |
|------|------|--------|
| 0:00 | Hook + who you are | Face / title |
| 0:15 | The problem + the SQL warehouse | Repo / architecture |
| 0:40 | KPIs (revenue, profit, margin) | Dashboard top |
| 0:55 | Live filtering | Click a filter |
| 1:10 | Insight: margin opportunity | Insights tab |
| 1:35 | Insight: retention | Insight cards |
| 1:50 | Outro + CTA | Face |

**Delivery tips:** smile on the first line, slow down on the numbers, and actually *click* things while you talk — movement keeps people watching. Keep it under 2:00; shorter is better than dragging.
